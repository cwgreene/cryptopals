Setup (python):
- `python3 -m venv .venv`
- `. activate.src`
- `pip3 install -r requirements.txt`
- `pytest`

Note if pytest fails, try running `rehash` to ensure that zsh
hasn't cached the path.

Setup (vscode)
- Run `Create Environment` from the action menu. Select `Venv`
- Run `Configure Tests`. Select `pytest`
